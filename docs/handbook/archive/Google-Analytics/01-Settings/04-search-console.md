# Google Search Console (구글 서치 콘솔)

## 1. Google Search Console (이하 서치 콘솔)의 정의

서치 콘솔은 구글의 웹마스터 도구로, 구글 검색 엔진이 사이트를 색인할 수 있도록 사이트맵을 등록하고, SEO를 위한 관리 기능을 제공하는 서비스입니다.

## 2. GA와 서치 콘솔을 연동해야 하는 이유
