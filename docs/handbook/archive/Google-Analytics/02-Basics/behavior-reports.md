# 참여보고서

## 1. 참여보고서란?

유저가 웹사이트 또는 모바일 어플리케이션에서 상호작용하는 것을 확인할 수 있는 보고서입니다. 이벤트를 사용해 유저가 다음과 같은 작업을 하는 것을 측정할 수 있습니다.

ex)

- 계정 잔액 확인할 때
- 페이지를 천천히 아래로 스크롤
